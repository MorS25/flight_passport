Identity and Authentication for UTM / U-Space ecosystem

This is a production instance of Flight Passport. This server is used to authenticate and issue credentials in the form of JWT tokens for different U-Space / UTM services.

What can I do with a Token?

Using a JWT token issued by this server, you can connect to a DSS instance, use it for applications like Flight Spotlight and across many other applications and serivice prodivers (e.g. query registration data).
